- Learnings:
  - Deep recursion for sanitization must traverse ALL values including safe fields — safe fields protect key names, not nested content
  - API key regex needs [A-Za-z0-9_-] (with underscore/dash) to catch patterns like sk_live_xxx
  - RegExp with /g flag needs lastIndex reset or new RegExp() for each test() call to avoid skipping matches
  - Buffer.byteLength(JSON.stringify(obj), 'utf-8') is the correct way to measure JSON byte size for context budgets
  - Priority-based trimming: remove lowest priority items first (P4→P3→P2), never touch P1 (identity, active errors)
  - Semantic compression (truncate long text to 200 chars + "...") reduces size before removing entire items
